## Инструкция по вводу в Microsoft Project

1. **Создай ресурсы** — вкладка *Resource Sheet*, введи имена и ставки (Standard Rate).
2. **Создай WBS** — 19 задач + 9 вех с вложенностью через *Indent / Outdent*.
3. **Укажи длительности** каждой задачи в колонке *Duration*.
4. **Проставь зависимости** — колонка *Predecessors*, тип связи FS (Finish-to-Start).
5. **Назначь ресурсы** — двойной клик по задаче → вкладка *Resources*.
6. **Добавь вехи** — для задач M1–M9 укажи Duration = 0 (◆ появится на диаграмме Ганта автоматически).
7. **Добавь повторяющиеся задачи** — *Task → Insert → Recurring Task*; укажи паттерн (Weekly / Bi-weekly), диапазон дат, назначь ресурсы.
8. **Установи крайние сроки**:
   - *Крайний срок окончания* — двойной клик → вкладка *Advanced* → поле **Deadline**.
   - *Крайний срок начала* — там же, **Constraint type**: *Start No Later Than* + дата.
9. **Установи дату начала** проекта: *Project → Project Information → Start date: 09.02.2026*.
10. **Укажи загрузку ресурсов** через *Resource Usage* — убедись, что нет перегрузок (красный индикатор).
11. **Сохрани базовый план** — *Project → Set Baseline → Set Baseline*.
12. **Просмотри критический путь** — *Format → Critical Tasks* (подсветятся красным).
13. **Проверь бюджет** — *Project → Reports → Costs → Budget Report*.
