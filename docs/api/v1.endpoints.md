# API v1: Актуальные endpoints

Снимок актуального набора endpoint-ов backend на `2026-04-14`.

## Health

- `GET /health`
- `GET /health/db`

## Auth

- `POST /api/v1/auth/login`
- `POST /api/v1/auth/register`
- `POST /api/v1/auth/refresh`
- `POST /api/v1/auth/logout`

## Справочники (CRUD)

- `/api/v1/user-roles`
- `/api/v1/application-statuses`
- `/api/v1/application-action-statuses`
- `/api/v1/topic-statuses`
- `/api/v1/topic-creator-types`
- `/api/v1/notification-types`
- `/api/v1/academic-degrees`
- `/api/v1/academic-titles`
- `/api/v1/positions`
- `/api/v1/study-groups`

Для каждого ресурса поддерживаются `GET list`, `GET by id`, `POST`, `PUT`, `PATCH`, `DELETE`.

## Доменный read API

- `GET /api/v1/teachers`
- `GET /api/v1/teachers/{id}`
- `GET /api/v1/students`
- `GET /api/v1/students/{id}`
- `GET /api/v1/topics`
- `GET /api/v1/topics/{id}`

## Topics

- `DELETE /api/v1/topics/{id}`
